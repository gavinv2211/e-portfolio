Seminar 4 - K-Means Clustering

Using the iris.csv dataset from UCI Machine Learning, perform K-means clustering, using 3 clusters

K-means clustering is a simple and popular unsupervised machine learning algorithm. A cluster refers to a collection of datapoints aggregated together, based on certain similarities, K refers to the number of cetroids to be used in the dataset (Education Ecosystem, 2018). A centroid is a location representing the center of a cluster (Education Ecosystem, 2018). Each data point is assigned to the nearest centroid based on the Euclidean distance method. Since there are 4 features in the Iris Dataset, each data point is represented as a 4-dimensional space.
