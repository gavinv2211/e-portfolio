Simple Perceptron

A simple perceptron is a very simple type of artificial neural network, used for binary classification and consists of a single layer of weights connecting inputs to a single output.

Key components include:

Inputs: The features of the data (The notebook exercise uses Age and Years of work experience)
Weights: Parameter values that gets adjusted during training.
Bias: The Bias is a constant added to the weighted sum to shift the activation function. (The notebook does not make use of this constant).
Activation Function: Determines the output. If the sum function greater or equal to 1 then output = 1 else output = 0 (Aka Step function).

The Sum Function:

\( z = \sum_{i=1}^{n} w_i \cdot x_i \)

As we increase the values of a weight in a perceptron, the output becomes more sensitive to changes in the corresponding input.
The step function output a binary value (0 or 1) small changes to \( x_i \) can cause larger changes in \(z\).

Perceptron AND operator

The AND operator takes two binary inputs and gives a single binary output. For the output to be 1, both inputs needs to be 1 else it will return 0

Multilayer perceptron

The Multilayer perceptron is a supervised feed forward neural network consisting of at least one input layer, a hidden layer and an output layer (Reddy, 2019). The hidden and output layer uses a nonlinear activation function.

The Multilayer perceptron notebook uses sigmoid as the activation function and introduces us to the concept of a Multilayer perceptron architecture. The Multilayer perceptron is a feed-forward neural-network which can handle the complexity of the XOR operator dataset. The XOR (Exclusive OR) takes two binary inputs just like the AND operator, but for the output to return 1, the inputs needs to differ. One of the inputs needs to be 1 and the other 0 if both are 1 or 0, then it will return 0.
