Jaccard Coefficient

For this exercise we were tasked to calculate the Jaccard Coefficient. The first Table is the original table. The second table was used to convert positives and negatives to binary classification. for the purposes of this exercise, values A & N are seen as negative.

The Jaccard Coefficient is a metric used to measure the similarity between sets (SetA and SetB).
Jaccard Coefficient: Intersect/(SetA + SetB - Intersect)

Jaccard Coefficient = 2/(2+3-2) = 0.67
