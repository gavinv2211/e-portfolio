Literature Review Outline

Focus, Aim and Audience:

Focus – The review will focus on using traditional machine learning models to predict sports results – specifically team sports (Rugby and Football). The aim is to identify which features are most important and how they vary between the sports.
Audience – The audience include data professionals with a keen interest in sports data, people betting on sports and bookmakers.

Significance:

Need and Significance – As data become more accessible, predictive models are becoming increasingly complex. While many studies focus on developing sophisticated models, fewer analyse which features contribute most to accurate predictions. Additionally, different team sports require different features for reliable forecasting.

Context and Perspective:

Context – The review is situated in the field of sports analytics and machine learning, focusing on predicting the outcomes on team sports using traditional machine learning methods. There is a gap in the literature regarding which features are most important and different features are used for different team sports.
Prespective – For data professionals it will provide guidance on feature selection. For sports bettors and bookmakers, it will offer insights into how machine learning models compare with bookmaker odds and whether certain features can provide an edge in forecasting results.

Framework:

Feature selection and importance – Review of studies analysing which features impact predictions the most. Comparison of feature importance across different sports.
Comparison of alternative approaches – How machine learning models compare to bookmaker odds. Identifying if there are any limitations to using traditional machine learning compared to deep learning approaches.

Strenths and limitations:

Strengths
There is a clear focus on Traditional machine learning models.
The review captures the diversity in predictive features across different team sports, highlighting how models need to be tailored to specific sports.

Limitations
By focusing on traditional machine learning techniques, the review may overlook more accurate techniques such as deep learning which have shown promise in sports prediction. The literature may not fully address the impact of data quality and availability which can affect model performance

Conclusions and Outcome:
The literature confirms that traditional machine learning models are still relevant for predicting sports results. Traditional machine learning techniques can offer a valuable alternative to more complex approaches like deep learning.

Too many features are available for sport predictions, while additional data may improve predictions, irrelevant or redundant features can introduce noise and decreased interpretability. Feature selection is important for accurate model development. Different team sports require different features for accurate predictability.
